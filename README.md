# vue-vs-react

vue vs react: reactivity

# Development scripts

### React App

> Run react app

```shell
$ cd react-app
$ pnpm install  # or $ pnpm i
$ pnpm run dev  # or $ pnpm dev
```

### Vue App

> Run vue app

```shell
$ cd vue-app
$ pnpm install  # or $ pnpm i
$ pnpm run dev  # or $ pnpm dev
```
